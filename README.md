# JRA馬券締切アラート (JRA Race Deadline Notifier)

JRA（日本中央競馬会）のレース締切時刻を通知するWebアプリケーションです。
Netkeibaのデータをリアルタイムで取得し、指定した条件（G1のみ、重賞のみなど）に基づいて予鈴を鳴らします。

## 機能 (Features)

*   **リアルタイムレース情報取得**: Netkeibaから当日のレース情報を自動取得
*   **重賞・G1モード**:
    *   **重賞モード**: G3以上のレースで、10分前に予鈴通知
    *   **G1限定モード**: G1レースのみ、10分前に予鈴通知
    *   **フィルター機能**: 平場・特別レースの直前通知（1分前）を無効化するオプション
*   **アラート通知**:
    *   **音声通知**: 締切2分前と10分前（重賞設定時）にアラート音を再生
    *   **ブラウザ通知**: デスクトップ通知で情報を表示（要権限許可）
*   **設定の保存**: ブラウザのローカルストレージに設定を保存
*   **PWA対応**: ホーム画面に追加してアプリのように利用可能
*   **リンク切り替え**: レース詳細へのリンクをNetkeiba（詳細）またはJRA公式（トップ）から選択可能

## セットアップと実行 (Setup & Run)

### 必要要件
*   Node.js 18+

### インストール
```bash
npm install
```

### 開発サーバーの起動
```bash
npm run dev
```
ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

### プロダクションビルド
```bash
npm run build
npm start
```
